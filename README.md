# Yulong Ye Homepage

## Structure

```text
_layouts/home.html              # Main page layout
_sass/_base.scss                # Shared base styles imported by Jekyll
assets/styles/                  # Compiled page stylesheets
assets/scripts/                 # Small browser scripts
assets/images/branding/         # University/brand images
assets/images/icons/            # Favicons
assets/images/profile/          # Profile photo
assets/images/publications/     # Publication teaser images
```

### Using Locally with Jekyll

You need Ruby and Bundler installed.

Install and run:

```bash
bundle install
bundle exec jekyll serve
```

Open <http://localhost:4000>.
